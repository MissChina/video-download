# M3U8 视频下载器

一款简单易用的 M3U8 视频下载工具，支持 HLS 流媒体下载和 AES-128 解密。

## 功能特点

- 支持 M3U8/HLS 播放列表下载
- 支持 AES-128 加密视频解密  
- 多线程并发下载
- 自动 FFmpeg 下载和管理
- 简洁直观的图形界面

## 安装和使用

### 环境要求

- Python 3.9+
- Windows 操作系统

### 安装依赖

```bash
pip install -r requirements.txt
```

### 运行程序

```bash
python main.py
```

## 使用说明

1. 在"下载"页面输入 M3U8 链接
2. 设置保存路径和文件名
3. 调整线程数（可选）
4. 点击"开始下载"

## 注意事项

- 首次使用需要在"设置"页面下载 FFmpeg（约70MB）
- 请确保有足够的磁盘空间
- 仅支持个人非商业使用
